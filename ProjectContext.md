The purpose of this project is to firstly allow me(the developer) to conduct quizzes online for my students.

Quiz Types:

- Live Quiz (Mostly MCQ/Identification)
- Pre Made
- Puzzles
- Programming Problems

Goal:

- Create an online platform where students can take a quiz and allows the intructor to save it later.
- Create a anti-cheating algorithm where as much as possible the students arent allowed to change tabs when taking a quiz, if so they can be penalized
- Allow instructors to easily record students quiz grades. From having a clean looking table for all the quiz results from sepcific quizzes or having the ability to export the results to apply in their own grading sheeet.
- Integrate AI to allow instructors to generate quizzes based on the lecture they have conducted.
  With this:
  - The AI should be able to read files (PDF, DOCX, ETC.).
  - The AI will generate a Quiz based on the instructors slides/document.
- Create a userfriendly interface where other instrcutors can use the website without having trouble navigating it
- Provide guides/manuals for unfamilliar users

Planned Features for sepcific users:

- Guests:
  - Instructors may require them to enter their fullname (This is to make instructors job easier)
  - Take a quiz using a code
  - Join a live quiz using a code
- Students:
  - Take a quiz (using a code or not)
  - Join a live quiz (using a code or not)
  - Can receive direct invites from instructors/other students
  - Track/Review quiz history
  -
- Instructors:
  - Upload slides/lecture document
  - Generate Quiz using the AI tool
  - Create a quiz
  - Generate a code/Invite link for the quiz
  - Track Students results
  - Export student results
